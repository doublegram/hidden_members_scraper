Doublegram Labs Hidden Members Scraper

This script allows you to scrape hidden members on groups. It will read the messages on the chat and then it will get the users data. You can connect multiple accounts to it, this way the data will be requested with all your connected accounts and you can decrease the number of total requests for each account. It will save a members.csv file compatible with Doublegram (Version >= 1.5.5).
V1.0

Prerequisites:

  • Python >= 3.8
  • PIP
You can refeer to Doublegram Pro docs to install Python and PIP

First steps:

  • Edit the configuration variables in start.py.

How to run:

  • Open the extracted folder of script in your terminal
  • Install the requirements with the command 
    pip install -r requirements.txt
  • Run the script by typing 
    python start.py
The commands may change based on your Python version and operating system. You might need to use 
python3
instead of 
python
or if you're using a virtual enviroment you have to use the path of your virtual enviroment as root path of pip and python, for example 
venvname/bin/python start.py
